# Subtec 608 closed captions unmute packet without active type packet L2 test

This python3 L2 test verifies behaviour of 608 type closed captions unmute 
packet without active type packet.

## Pre-requisites to L2 tests:

Subtec dependencies installed with *install_subtec_dependencies.sh* script
and subtec running.

## Run test using script:

From the *subtec-app/test/* folder run:

./run_l2_tests.py -i 1029
